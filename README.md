# Medieval Fantasy Side Scroller

A 2D platformer side-scroller game built with Pygame.

## How to Play

### 1. Setup the Environment
First, ensure you have Python 3 and Poetry installed. Then, use Poetry to install the project dependencies:
```bash
poetry install
```

### 2. Run the Game
Execute the main file using Poetry to start the game:
```bash
poetry run python main.py
```

### Controls
* **Left/Right Arrows**: Move player / Select Character
* **Up Arrow or Spacebar**: Jump
* **X Key**: Shoot projectile
* **Enter**: Confirm selection in menus
* **Escape**: Return to previous menu / Map
